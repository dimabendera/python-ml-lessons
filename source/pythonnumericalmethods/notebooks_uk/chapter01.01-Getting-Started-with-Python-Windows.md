<h2>Налаштування робочого середовища у Windows<a href="#set-up-working-environment-on-windows" title="Постійне посилання на цей заголовок"></a></h2>
<p>Перш ніж ми почнемо використовувати Python, нам потрібно налаштувати робоче середовище Python на комп'ютері. У цьому розділі ми ознайомимося з процесом його налаштування.</p>
<p>Існують різні способи встановлення Python та пов'язаних з ним пакетів. Ми рекомендуємо використовувати <a href="https://www.anaconda.com/download/">Anaconda</a> або <a href="https://conda.io/miniconda.html">Miniconda</a> для встановлення та керування пакетами. Залежно від вашої <em>операційної системи</em> (ОС), тобто Windows, Mac OS X або Linux, вам потрібно завантажити відповідний інсталятор для вашого пристрою. І Anaconda, і Miniconda мають на меті забезпечити прості способи управління робочим середовищем Python для наукових обчислень та наук про дані.</p>
<p>Тут ми будемо використовувати Windows як приклад, щоб показати вам процес встановлення. Користувачам Mac та Linux слід ознайомитися з главою 1.1 для повного опису процесів. Основні відмінності між Anaconda та Miniconda полягають у наступному:</p>
<ul>
<li><p><strong>Anaconda</strong> — це повний дистрибутив, що включає інтерпретатор Python, менеджер пакетів, а також найпоширеніші пакети для наукових обчислень.</p></li>
<li><p><strong>Miniconda</strong> — це полегшена версія Anaconda, яка не містить поширених пакетів, тому вам доведеться встановлювати всі необхідні пакети самостійно. Однак вона включає інтерпретатор Python та менеджер пакетів.</p></li>
</ul>
<p>Варіант, який ми тут обираємо, — це використання Miniconda для керування встановленням пакетів. Таким чином, ми можемо встановити лише ті, які нам потрібні.</p>

<p><strong>Використання оболонки Python або Ipython</strong></p>
<p>Найпростіший спосіб запустити код Python — через оболонку Python або Ipython (що означає Interactive Python). Оболонка Ipython багатша за оболонку Python, вона має такі функції, як автодоповнення по клавіші Tab, підсвічування повідомлень про помилки кольором, базова інтеграція з оболонкою UNIX тощо. Оскільки ми щойно встановили Ipython, давайте спробуємо запустити з його допомогою приклад "hello world". Щоб запустити оболонку Python або Ipython, потрібно ввести її назву в терміналі (див. малюнок нижче). Потім ми можемо виконувати команди Python, вводячи їх в оболонку та натискаючи <code><span>Enter</span></code>, і одразу бачити результат.
