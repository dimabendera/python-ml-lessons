<h1>Python як калькулятор<a href="#python-as-a-calculator" title="Постійне посилання на цей заголовок"></a></h1>
<p>Ми познайомимо вас з Python, продемонструвавши можливості, які є в будь-якому стандартному графічному калькуляторі. <strong>Арифметична операція</strong> — це додавання, віднімання, множення, ділення або піднесення до степеня між двома числами. <strong>Арифметичний оператор</strong> — це символ, який Python зарезервував для позначення однієї з вищезгаданих операцій. Цими символами є + для додавання, - для віднімання, * для множення, / для ділення та ** для піднесення до степеня.</p>
<p>Ми кажемо, що інструкція або операція <strong>виконується</strong>, коли вона обробляється комп'ютером. Інструкція виконується в командному рядку шляхом її введення там, де ви бачите символ &gt;&gt;&gt; в оболонці Python (або знак <code>In [1]:</code> в Ipython), а потім натискання <code>Enter</code>. Або введіть її у клітинку коду в Jupyter notebook і натисніть <code>Shift + Enter</code>. Оскільки ми будемо використовувати Jupyter notebook далі, тут ми покажемо всі приклади в оболонці Ipython (оболонка Python схожа), щоб ви могли ознайомитися з різними варіантами.</p>
<p><strong>СПРОБУЙТЕ!</strong> Обчисліть суму 1 та 2.</p>
<pre><code>In [1]: 1 + 2
Out[1]: 3
</code></pre>
<p><strong>Порядок операцій</strong> — це стандартний порядок пріоритету, який різні операції мають по відношенню одна до одної. Python використовує той самий порядок операцій, який ви вивчали в школі. Піднесення до степеня виконується перед множенням і діленням, які виконуються перед додаванням і відніманням. Дужки, (), також можуть використовуватися в Python для перевизначення стандартного порядку операцій.</p>
<p><strong>СПРОБУЙТЕ!</strong> Обчисліть \(\frac{3*4}{(2^2+4/2)}\).</p>
<pre><code>In [2]: (3*4)/(2**2 + 4/2)
Out[2]: 2.0
</code></pre>
<p><strong>ПОРАДА!</strong> Ви могли помітити, що <code>Out[2]</code> є результуючим значенням останньої виконаної операції. Ви можете використовувати символ _, щоб представити цей результат для розбиття складних виразів на простіші команди.</p>
<p><strong>СПРОБУЙТЕ!</strong> Обчисліть 3, поділене на 4, потім помножте результат на 2, а потім піднесіть результат до 3-го степеня.</p>
<pre><code>In [3]: 3/4
Out[3]: 0.75

In [4]: _*2
Out[4]: 1.5

In [5]: _**3
Out[5]: 3.375
</code></pre>
<p>Python має багато базових арифметичних функцій, таких як <code>sin</code>, <code>cos</code>, <code>tan</code>, <code>asin</code>, <code>acos</code>, <code>atan</code>, <code>exp</code>, <code>log</code>, <code>log10</code> та <code>sqrt</code>, що зберігаються в модулі (ми пояснимо це пізніше в цьому розділі), який називається <strong>math</strong>. Ми можемо спочатку імпортувати цей модуль, щоб отримати доступ до цих функцій.</p>
<pre><code>In [6]: import math
</code></pre>
<p><strong>ПОРАДА!</strong> У Jupyter notebook та Ipython ви можете швидко переглянути вміст модуля, набравши <code>назва модуля + крапка + TAB</code>. Крім того, якщо ви наберете перші кілька літер назви функції та натиснете TAB, вона може автоматично доповнити її за вас, так зване автодоповнення по TAB.</p>
<p><img alt="Автодоповнення по TAB" src="/var/www/projects_teacher/INTITA-AI-ML/source/pythonnumericalmethods/_images/01.02.01-ipython_tab.png"/></p>
<p>Спосіб використання цих математичних функцій — <code>модуль.функція</code>, вхідні дані для них завжди розміщуються всередині дужок, пов'язаних з назвою функції. Для тригонометричних функцій корисно мати доступ до значення \(\pi\). Ви можете викликати це значення в будь-який час, набравши <code>math.pi</code> у клітинці коду. Зауважте, що значення \(\pi\) зберігається в Python з точністю до 16 знаків.</p>
<p><strong>СПРОБУЙТЕ!</strong> Знайдіть квадратний корінь з 4.</p>
<pre><code>In [7]: math.sqrt(4)
Out[7]: 2.0
</code></pre>
<p><strong>СПРОБУЙТЕ!</strong> Обчисліть \(sin(\frac{\pi}{2})\).</p>
<pre><code>In [8]: math.sin(math.pi/2)
Out[8]: 1.0
</code></pre>
<p>Python компонує функції так, як ви очікуєте, причому першою виконується найвнутрішня функція. Те саме стосується викликів функцій, які поєднуються з арифметичними операціями.</p>
<p><strong>СПРОБУЙТЕ!</strong> Обчисліть \(e^{log10}\).</p>
<pre><code>In [9]: math.exp(math.log(10))
Out[9]: 10.000000000000002
</code></pre>
<p>Зауважте, що функція <code>log</code> в Python — це \(log_e\), або натуральний логарифм. Це не \(log_{10}\). Якщо ви хочете використовувати \(log_{10}\), вам потрібно використовувати <code>math.log10</code>.</p>
<p><strong>ПОРАДА!</strong> Ви можете бачити, що результат вище повинен бути 10, але він показує 10.000000000000002, це пов'язано з апроксимацією чисел в Python, про що ми дізнаємося більше в розділі 9.</p>
<p><strong>СПРОБУЙТЕ!</strong> Обчисліть \(e^\frac{3}{4}\)</p>
<pre><code>In [10]: math.exp(3/4)
Out[10]: 2.117000016612675
</code></pre>
<p><strong>ПОРАДА!</strong> Використання СТРІЛКИ ВГОРУ в командному рядку викликає попередні виконані команди. Якщо ви випадково ввели команду неправильно, ви можете використати СТРІЛКУ ВГОРУ, щоб викликати її, а потім відредагувати, замість того, щоб передруковувати весь рядок.</p>
<p>Багато разів, коли ви використовуєте функцію в Python, вам може знадобитися допомога щодо її контексту. В Ipython або Jupyter notebook ви можете переглянути опис будь-якої функції, набравши <code>function?</code>, тобто знак питання є швидким способом отримання довідки. Якщо ви бачите незнайому функцію, гарною практикою буде використати знак питання, перш ніж запитувати у викладачів, що робить конкретна функція.</p>
<p><strong>УВАГА!</strong> Для деяких функцій файл довідки може бути надзвичайно складним і багатослівним, навіть для простих функцій. У таких випадках не бійтеся звертатися за допомогою до свого викладача.</p>
<p><strong>СПРОБУЙТЕ!</strong> Використайте знак питання, щоб знайти визначення функції факторіала</p>
<pre><code>In [11]: math.factorial?
</code></pre>
<pre>Сигнатура: math.factorial(x, /)
Рядок документації:
Знайти x!.

Викликати ValueError, якщо x є від'ємним або нецілим.
Тип:      вбудована функція або метод
</pre>
<p>Python викличе помилку <code>ZeroDivisionError</code>, коли ви матимете вираз 1/0, що є нескінченністю, щоб нагадати вам.</p>
<pre><code>In [12]: 1/0
</code></pre>
<pre>---------------------------------------------------------------------------
ZeroDivisionError                         Traceback (most recent call last)
&lt;ipython-input-12-9e1622b385b6&gt; in &lt;module&gt;()
----&gt; 1 1/0

ZeroDivisionError: division by zero
</pre>
<p>Ви можете ввести <code>math.inf</code> в командному рядку, щоб позначити нескінченність, або <code>math.nan</code>, щоб позначити щось, що не є числом, але ви хочете, щоб воно оброблялося як число. Якщо це вас плутає, цю відмінність можна поки пропустити; її буде пояснено чіткіше, коли це стане важливим. Нарешті, Python також може обробляти уявні числа.</p>
<p><strong>СПРОБУЙТЕ!</strong> \(1/\infty\), та \(\infty*2\) щоб перевірити, що Python обробляє нескінченність так, як ви очікуєте.</p>
<pre><code>In [13]: 1/math.inf
Out[13]: 0.0

In [14]: math.inf * 2
Out[14]: inf
</code></pre>
<p><strong>СПРОБУЙТЕ!</strong> Обчисліть \(\infty/\infty\)</p>
<pre><code>In [15]: math.inf/math.inf
Out[15]: nan
</code></pre>
<p><strong>СПРОБУЙТЕ!</strong> Обчисліть суму 2 + 5i</p>
<pre><code>In [16]: 2 + 5j
Out[16]: (2+5j)
</code></pre>
<p>Зауважте, що в Python для представлення уявної частини використовується j замість i.</p>
<p>Інший спосіб представити комплексне число в Python — це використати функцію complex.</p>
<pre><code>In [17]: complex(2,5)
Out[17]: (2+5j)
</code></pre>
<p>Python також може обробляти наукову нотацію, використовуючи літеру e між двома числами. Наприклад, \(1e6=1000000\) та \(1e-3=0.001\).</p>
<p><strong>СПРОБУЙТЕ!</strong> Обчисліть кількість секунд у 3 роках, використовуючи наукову нотацію.</p>
<pre><code>In [18]: 3e0*3.65e2*2.4e1*3.6e3
Out[18]: 94608000.0
</code></pre>
<p><strong>ПОРАДА!</strong> Кожного разу, коли ми вводимо функцію з модуля <em>math</em>, ми завжди пишемо <code>math.function_name</code>. Як альтернатива, є простіший спосіб, наприклад, якщо ми хочемо використовувати <code>sin</code> та <code>log</code> з модуля <code>math</code>, ми можемо імпортувати їх так: <code>from math import sin, log</code>. Тоді все, що вам потрібно робити при використанні цих функцій, — це викликати їх безпосередньо, наприклад, <code>sin(20)</code> або <code>log(10)</code>.</p>
<h2>Основні типи даних<a href="#basic-data-types" title="Постійне посилання на цей заголовок"></a></h2>
<p>Ми щойно навчилися використовувати Python як калькулятор для роботи з різними значеннями даних. У Python є кілька типів даних, які нам потрібно знати; для числових значень це типи int, float та complex.</p>
<ul>
<li><p><strong>int</strong>: Цілі числа, такі як 1, 2, 3, …</p></li>
<li><p><strong>float</strong>: Числа з плаваючою комою, такі як 3.2, 6.4, …</p></li>
<li><p><strong>complex</strong>: Комплексні числа, такі як 2 + 5j, 3 + 2j, …</p></li>
</ul>
<p>Ви можете використовувати функцію <em>type</em> для перевірки типу даних різних значень.</p>
<p><strong>СПРОБУЙТЕ!</strong> Дізнайтеся тип даних для 1234.</p>
<pre><code>In [19]: type(1234)
Out[19]: int
</code></pre>
<p><strong>СПРОБУЙТЕ!</strong> Дізнайтеся тип даних для 3.14.</p>
<pre><code>In [20]: type(3.14)
Out[20]: float
</code></pre>
<p><strong>СПРОБУЙТЕ!</strong> Дізнайтеся тип даних для 2 + 5j.</p>
<pre><code>In [21]: type(2 + 5j)
Out[21]: complex
</code></pre>
<p>Звісно, існують і інші типи даних, такі як булевий, рядковий тощо, ми познайомимося з ними пізніше в цій книзі.</p>
<p>У цьому розділі ми говорили про те, як використовувати Python як калькулятор, виконуючи команди в оболонці Python. Давайте продовжимо і дізнаємося більше про керування пакетами, тобто як встановлювати, оновлювати та видаляти пакети, перш ніж ми перейдемо до написання коду.</p>
