<h1>Задачі<a href="#problems" title="Permalink to this headline"></a></h1>
<ol>
<li><p>Використайте розклади в ряд Тейлора, щоб показати, що <span>\(e^{ix} = \cos(x) + i\sin(x)\)</span>, де <span>\(i = \sqrt{-1}\)</span>.</p></li>
<li><p>Використайте лінійну апроксимацію <span>\(\sin(x)\)</span> навколо <span>\(a = 0\)</span>, щоб показати, що <span>\(\frac{\sin(x)}{x} \approx 1\)</span> для малих <span>\(x\)</span>.</p></li>
<li><p>Напишіть розклад у ряд Тейлора для <span>\(e^{x^2}\)</span> навколо <span>\(a = 0\)</span>. Напишіть функцію <em>my_double_exp(x, n)</em>, яка обчислює апроксимацію <span>\(e^{x^2}\)</span> за допомогою перших <span>\(n\)</span> членів розкладу в ряд Тейлора. Переконайтеся, що <em>my_double_exp</em> може приймати масиви як вхідні дані.</p></li>
<li><p>Напишіть функцію, яка надає апроксимацію функції *np.exp* за допомогою ряду Тейлора навколо 0 для порядку від 1 до 7. Обчисліть межу похибки усічення для порядку 7.</p></li>
<li><p>Обчисліть розклад Тейлора четвертого порядку для <span>\(\sin(x)\)</span> та <span>\(\cos(x)\)</span> та <span>\(\sin(x)\cos(x)\)</span> навколо 0. Що дає меншу похибку для <span>\(x = \pi/2\)</span>: обчислення розкладу Тейлора для <span>\(\sin\)</span> та <span>\(\cos\)</span> окремо з подальшим множенням результатів, чи обчислення розкладу Тейлора для добутку спочатку, а потім підстановка <span>\(x\)</span>?</p></li>
<li><p>Використайте ряд Тейлора 4-го порядку для апроксимації <span>\(\cos(0.2)\)</span> та визначте межу похибки усічення.</p></li>
<li><p>Напишіть функцію <em>my_cosh_approximator(x, n)</em>, де вихідним значенням є апроксимація гіперболічного косинуса <span>\(\text{cosh}(x)\)</span> за допомогою ряду Тейлора <span>\(n\)</span>-го порядку, взятого навколо <span>\(a = 0\)</span>. Можна припустити, що <span>\(x\)</span> є масивом, а <span>\(n\)</span> — додатним цілим числом (включаючи 0). Нагадаємо, що
$<span>\(
{\rm cosh}(x) = (e^x + e^{-x})/2.
\)</span><span>\(
Попередження: Апроксимації для \)</span>n = 0<span>\( та \)</span>n = 1<span>\( будуть еквівалентними, апроксимації для \)</span>n = 2<span>\( та \)</span>n = 3$ будуть еквівалентними, і так далі.</p></li>
</ol>
