<h1>Постановка крайової задачі для ЗДР<a href="#ode-boundary-value-problem-statement" title="Постійне посилання на цей заголовок"></a></h1>
<p>У попередньому розділі ми говорили про задачі з початковими умовами для звичайних диференціальних рівнянь. Ми бачимо, що в задачах з початковими умовами всі відомі значення задаються для одного й того ж значення незалежної змінної, зазвичай на нижній межі інтервалу, звідси й походить термін «початкові». Тоді як у цьому розділі ми обговоримо інший тип задач — <strong>крайові задачі</strong>, як випливає з назви, відомі значення задаються на крайніх значеннях незалежної змінної, тобто на межах інтервалу.</p>
<p>Наприклад, якщо ми маємо просте звичайне диференціальне рівняння 2-го порядку,</p>

\[\frac{d^2f(x)}{dx^2} = \frac{df(x)}{dx} + 3\]
<p>якщо незалежна змінна знаходиться в області [0, 20], задача з початковими умовами матиме дві умови для значення 0, тобто ми знаємо значення <span>\(f(0)\)</span> та <span>\(f'(0)\)</span>. Натомість, крайові задачі задаватимуть значення для <span>\(x = 0\)</span> та <span>\(x = 20\)</span>. Зауважте, що для розв'язання ЗДР першого порядку для отримання часткового розв'язку нам потрібне одне обмеження, тоді як для ЗДР <span>\(n\)</span>-го порядку нам потрібні <span>\(n\)</span> обмежень.</p>
<p>Постановка крайової задачі для звичайного диференціального рівняння <span>\(n\)</span>-го порядку формулюється так:</p>

\[
F\left(x, f(x), \frac{df(x)}{dx}, \frac{d^2f(x)}{dx^2}, \frac{d^3f(x)}{dx^3}, \ldots, \frac{d^{n-1}f(x)}{dx^{n-1}}\right) = \frac{d^{n}f(x)}{dx^{n}},
\]
<p>Щоб розв'язати це рівняння на інтервалі <span>\(x\in[a, b]\)</span>, нам потрібні <span>\(n\)</span> відомих крайових умов у точках <span>\(a\)</span> та <span>\(b\)</span>. Для випадку 2-го порядку, оскільки крайова умова може бути або значенням f(x), або значенням похідної <span>\(f'(x)\)</span>, ми можемо мати кілька різних випадків для заданих значень. Наприклад, значення крайових умов можуть бути задані як:</p>
<ol>
<li><p>Задано два значення <span>\(f(x)\)</span>, тобто відомі <span>\(f(a)\)</span> та <span>\(f(b)\)</span></p></li>
<li><p>Задано дві похідні <span>\(f'(x)\)</span>, тобто відомі <span>\(f'(a)\)</span> та <span>\(f'(b)\)</span></p></li>
<li><p>Або відомі змішані умови з двох вищезгаданих випадків, тобто відомі або <span>\(f(a)\)</span> та <span>\(f'(b)\)</span>, або <span>\(f'(a)\)</span> та <span>\(f(b)\)</span>.</p></li>
</ol>
<p>У будь-якому випадку, щоб отримати частковий розв'язок, нам потрібні дві крайові умови. Крайова задача для ЗДР другого порядку також називається двоточковою крайовою задачею. Задачі для ЗДР вищих порядків потребують додаткових крайових умов, зазвичай значень вищих похідних незалежних змінних. У цьому розділі зосередимося на двоточкових крайових задачах.</p>
<p>Розглянемо приклад крайової задачі та побачимо, як ми можемо її розв'язати в наступних розділах.</p>

<p>Ребра використовуються в багатьох застосуваннях для збільшення тепловіддачі від поверхонь. Зазвичай конструкція охолоджувальних штифтових ребер зустрічається в багатьох застосуваннях, наприклад, штифтове ребро, що використовується як радіатор для охолодження об'єкта. Ми можемо змоделювати розподіл температури в штифтовому ребрі, як показано на малюнку вище, де довжина ребра становить <span>\(L\)</span>, а початок і кінець ребра — <span>\(x=0\)</span> та <span>\(x=L\)</span> відповідно. Температура на двох кінцях становить <span>\(T_0\)</span> та <span>\(T_L\)</span>. <span>\(T_s\)</span> — це температура навколишнього середовища. Якщо ми враховуємо як конвекцію, так і випромінювання, стаціонарний розподіл температури штифтового ребра <span>\(T(x)\)</span> можна змоделювати за допомогою наступного рівняння:</p>

\[\frac{d^2T}{dx^2} - \alpha_1(T - T_s) - \alpha_2(T^4 - T^4) = 0\]
<p>з крайовими умовами: <span>\(T(0) = T_0\)</span> та <span>\(T(L) = T_L\)</span>, де <span>\(\alpha_1\)</span> та <span>\(\alpha_2\)</span> — коефіцієнти. Це ЗДР другого порядку з двома крайовими умовами, отже, ми можемо розв'язати його, щоб отримати часткові розв'язки.</p>
<p>Решта цього розділу охоплює два методи чисельного наближення розв'язку крайових задач на числовій сітці. Ми розглянемо методи стрільби та методи скінченних різниць для розв'язання крайових задач для ЗДР.</p>
